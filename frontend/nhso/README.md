
## Description
This is a web application for NHSO.


## Installation

**1. Node version**
```
Compatible node version is 14.15
```

**2. Set up env**
```
REACT_APP_BASE_URL={backend-url-endpoint}
```
**3. Install dependencies**
```
yarn
```
**4. Build project**
```
yarn build
```
**5. Install serve**
```
yarn install -g serve
```
**6. Start server**
```
serve -s build -l 8080
```
